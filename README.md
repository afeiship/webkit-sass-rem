# webkit-sass-rem
> Rem solution based on vw and media query.

## usage
+ https://afeiship.github.io/webkit-sass-rem/

```scss
@import 'node_modules/webkit-sass-rem/dist/webkit-sass-rem.scss';

// based: 750px 
// 13.33vw = 100px

// based: 375px 
// 13.33vw = 50px
```
